# SuperJump-Patch

Мод-патч для мини-игры "Прятки" на сервере TeslaCraft, который возвращает старую
механику супер-прыжков. (1.12.2 Forge)

## Как работает

Когда в руке держишь алмазный топор с "Топор охотника" в названии, клик правой
кнопкой мыши даёт супер-прыжок. На любых других предметах ничего не происходит.

## Сборка

```
./gradlew build
```

Готовый jar появится в `build/libs/`.

## Иконка

Положи иконку 128x128 в `src/main/resources/assets/superjumppatch/icon.png`.
